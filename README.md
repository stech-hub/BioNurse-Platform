# BioNurse Platform

**BioNurse** is a free nursing education platform providing:

- Free scholarships (Liberia & online)
- Free online courses for nursing
- Nursing notes & procedure guides
- Over 1000 quizzes and tests with certificates
- Support / Donations section (MoMo, Bank, WhatsApp)
- Achievements & badges system
- APK download link: [Download MyApp](https://github.com/stech-hub/bionurseapk-website/releases/download/v1/myapp.apk)

## Tech Stack

- Next.js (React)
- Tailwind CSS
- jsPDF (for certificates)
- Vercel Deployment Ready

## Deployment

To deploy locally:

```bash
npm install
npm run dev
```

Visit `http://localhost:3000`.

To deploy on **Vercel**, simply import the repository and deploy. Vercel auto-detects Next.js apps.
